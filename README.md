# jean-philippe

The intuitive, web-based menu and ordering management application for restaurants.

## Platform

- Java 17
- Maven 3.8.3
- Vaadin 21
- Java Spring Boot 2.5.6
- MariaDB
- node v14.18.2

## How to run

```bash
mvn clean compile spring-boot:run -Pproduction
```
